# github

Sub-modules for the `GitHub` class.

| File               | Purpose                             |
|--------------------|-------------------------------------|
| `rest.cr`          | HTTP REST client for the GitHub API |
| `rest/`            | REST sub-modules (paginator, etc.)  |
| `option_parser.cr` | CLI option parsing and commands     |
| `api.json`         | GitHub REST API OpenAPI 3.1 spec    |
